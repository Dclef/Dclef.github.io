科学上网的快速搭建

小白入门 2019/11/02

先说VPN原理：通常来讲以流量的方式将数据传输到代理商服务器在通过代理商加密传输过来。

简单来讲墙是一道在你们中间的大闸门，你对着 google 叫 google，你问什么是VPN，闸门听见了，说 wtf 你还敢叫 google，就把闸门关了，你的信息就传不到 google 那里了，于是因为 google 完全听不到你的声音，自然也就不能告诉你：VPN是什么。

1.下载个SSR，也就是我们俗称的飞机场。

链接：https://pan.baidu.com/s/12qs4ydb9dVS-5v43DlXKYA
提取码：3syu

2.安装及配置
windows直接解压即可，手机安装软件也即可。
windows
解压打开这个
这时候打开网站注册账号

https://www.clashcloud.net/#/auth/register?code=wU4f

)注册完毕以后点击购买套餐

购买完毕后打开用户中心

点击链接，复制链接后用鼠标右键点击右下角

点击此项你便成功科学上网！当然代理规则改为绕过局域网和大陆，这样你打开本地的网站不会卡.第一次一个月9块。20G我觉得够用了

android
打开你的SSR，点击标题

点击右下角的加号，添加你的订阅地址


或者扫描二维码都可以

同样的代理规则改为绕过局域网和大陆

这样你便会享受墙外的乐趣和减少墙流量的使用。比如上油管或者某P站和电报。
