hexo博客主题搭建

 · 2019-12-10  ·次阅读

搭建完以后你会发现hexo的初始主题有些难看，所以我们选择修改主题。
这里只讲next主题修改
下载next

先进入BLOG在themes文件夹里右键gitbash进入并输入

git clone https://github.com/iissnan/hexo-theme-next themes/next

修改主题
在config.yml找到themes，这里可直接使用ctrl+shift+f快速查找
找到themses时，lanspeas修改为next
然后在blog文件夹右键gitbash输入

hexo clean
hexo g -d

然后你就可以在你的入http://yourname.github.io看到你的新主题了

注意事项
主题里也有个config.yml你需要修改
站点config.yml

title: 老猫的博客 #网站标题
subtitle: #网站副标题
description: #网站描述
author: dclef #您的名字
language: zh-Hans #网站使用的语言
timezone: #网站时区。Hex 默认使用您电脑时区。

创建分类标签归档关于

hexo new page categories
hexo new page tags
hexo new page about
hexo new page archives

分别在分类标签归档关于里的index文件中添加

type: “categories”
type: “tags”
type: “about”
type: “archives”
